Command

apt apt-get

apt = apt-get + apt-cache + apt-config

简单来说就是：apt = apt-get、apt-cache 和 apt-config 中最常用命令选项的集合。

常用命令

  • 为文件目录树全部赋予权限：chmod 775 -R filename
  • 迭代删除文件夹中所有内容：rm -rf &lt;filename or dirname&gt;

挂载数据盘

  • 查看已有磁盘：fdisk -l
  • 挂载磁盘：mount /dev/vdb1 /home/data
